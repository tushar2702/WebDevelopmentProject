I used a few of my experiments CSS which helped me make the rounded text boxes and give shadows using box-shadow property of CSS3. So it was a lot of reuse of my existing knowledge of CSS.

Now, for the Check-in date and Check-out date, I wanted to use a datepicker and therefore Jquery's datepicker was the best to use. So, I added the jquery ui library through google apis. I thought this was pretty easy and would work in the first try itself. But, it didn't. This is because there are problems when we use asp:textbox in child pages of existing master pages. The child page controls get strange id names and hence instead of using #id_name I need to use input[id$='id_name']. Or alternatively, I can use HTML text box rather than ASP.NET text boxes. And that worked!!! It was this link that helped me understand why such problems occur http://forum.jquery.com/topic/jquary-datepicker-is-not-working-inside-asp-net-masterpage-application.
