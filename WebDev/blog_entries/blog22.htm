After the "Accomodations" page for my project was done, I started working on the "Services" page that are the services offered by my hotel. This includes a few categories like the swimming pool, fitness centre, spa etc. and I was thinking of a way to represent these and I didn't want to use Multiviews since I had already used it for the "Accomodations" page.

So, I thought of using images for each category and a small description. Now if you want to explore a particular service in detail, you would have to click on the respective image or link and go to that page for further details. It's quite simple but interesting.

Each image in the "Services" page is floating left and this makes me arrange all the image div's as I want. I have used basic CSS to make this!!
